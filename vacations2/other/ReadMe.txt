- установить glassfish 4
- настроить в админку по скринам: 
		- имя JDBC-ресурса: vacRes
		- JDBC Connection pools: vac
		- настройки к базе: свои
		- у пользователя должны быть проставлена база по умолчанию наша
- скопировать sqljdbc4.jar в C:\glassfish4\glassfish\domains\domain1\lib\ext
- перезапустить гласфиш
- пропинговать для проверки
- выполнить скрипт для обновления базы: UpdateExistDB.sql
- задеплоить war-архив
